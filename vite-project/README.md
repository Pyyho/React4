# Приложение "Список заметок"

Простое и удобное React-приложение для управления текстовыми заметками.

## Функциональность

- ✅ Добавление новых текстовых заметок
- ✅ Просмотр списка всех заметок
- ✅ Удаление заметок
- ✅ Автоматическое сохранение в localStorage
- ✅ Адаптивный дизайн
- ✅ Валидация ввода

## Запуск приложения

```bash
npm install
npm run dev
